# 开单导出模板锁版规则（Excel/PDF统一）

版本：v1（2026-02-19）
适用：`src/routes/workOrders.js` 的 Excel HTML 与 PDF 绘制逻辑
目标：后续只做局部改动，不再破坏整体版式

---

## 一、总原则

1. **Excel 与 PDF 同步改动**：任何字段位置变更必须双端同时修改。  
2. **只改明确授权区域**：未在“可改区”内的字段，禁止改宽度/列数。  
3. **同字段单一主位置**：避免重复展示导致冲突（如订单数量、交货日期、压辊）。  
4. **优先保证表头/分区稳定**：标题居中、分区顺序固定，不做临时拼接。

---

## 二、固定区（禁止改宽/改列）

### 1) 表头区
- 标题：`潮安区华胜印刷有限公司生产工作表`
- 右上角单独显示：`压辊：<数字>`
- 禁止把压辊并入标题文案

### 2) 基础信息区
- 日期/业务员/编号
- 客户/订单ID/规格
- 品名单独大行
- 禁止新增重复“订单数量”

### 3) 印膜信息区
- 参考色、油墨要求、交货日期同一行
- 印膜大行
- 订单数量/开单数量/印刷班次一行
- 备注大行

### 4) 覆膜工艺区（当前锁版）
- 分区名固定：`覆膜工艺`
- 第一层/第二层/第三层三行结构固定（层级文字必须居中）
- 规格与数量（重量）分列，禁止再合并到同一格
- 覆膜要求两行合并仅允许发生在左侧说明区
- 严禁改动第一/二/三层对应列宽（仅允许调行高）

### 5) 制袋与交付区
- 袋形/是否外加工
- 拉链位置/撕口位置/挂孔位置
- 孔位/封边/其它要求
- 压辊不在本区展示

---

## 三、可改区（允许局部调优）

1. 字号微调（±1号）
2. 文案统一（复膜→覆膜）
3. 行高微调（不改列宽）
4. 对齐方式（仅当前单元格，不改邻列宽）

---

## 四、变更流程（强制）

每次改动前后都要执行：

1. 记录“改动目标字段 + 所在分区”
2. 同步修改 Excel + PDF
3. 检查以下四项：
   - 表头是否仍居中
   - 覆膜工艺右侧三层是否等宽对齐
   - 制袋与交付是否未被上方挤压
   - 订单数量是否只保留指定位置
4. 重启并健康检查：`/health = 200`

---

## 五、当前用户明确口径（以用户最新指令为准）

1. 只在表头右上角显示压辊（数字可放大，标签不放大）
2. 制袋与交付中不显示压辊
3. 覆膜工艺命名统一，不再使用“复膜结构/复膜类型”混用
4. 规格和数量（重量）必须拆列显示

---

> 后续如果用户口径变更，先更新本文件再改代码，避免回归错误。
